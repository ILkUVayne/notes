# go 环境配置

## 相关工具安装

- goimports-reviser

~~~bash
go install github.com/incu6us/goimports-reviser/v3@latest
~~~

- gofumpt

~~~bash
go install mvdan.cc/gofumpt@latest
~~~

- golines

~~~bash
go install github.com/segmentio/golines@latest
~~~

- go-delve

~~~bash
go install github.com/go-delve/delve/cmd/dlv@latest
~~~
