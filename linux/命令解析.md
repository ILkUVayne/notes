# linux命令解析

~~~bash
$ ffmpeg -i /mnt/f/video/1251203951-1-192.mp4 -hide_banner 2>&1 | grep -m 1 'Duration:' | cut -d ' ' -f 4 | sed s/,// | awk -F ":" '{print(($1 * 3600) + ($2 * 60 + $3)) * 1000}'

# 1. ffmpeg -i /mnt/f/video/1251203951-1-192.mp4 -hide_banner
# 获取视频元数据信息

# 2. 2>&1
# 将标准错误输出重定向到标准输出，用于后续的过滤查找等操作

# 3. grep -m 1 'Duration:'
# 过滤包含'Duration:'的数据，'-m 1' 表示在查到的第一行停止，即只查询一行数据

# 4. cut -d ' ' -f 4
# cut 命令在Linux和Unix中的作用是从文件中的每一行中截取出一些部分，并输出到标准输出中
# -d 表示指定分隔符， -f 表示提取指定的字段
# -d ' ' -f 4  表示获取每行以' '分隔开的第4个字段

# 5 sed s/,//
# 替换逗号

# 6. awk -F ":" '{print(($1 * 3600) + ($2 * 60 + $3)) * 1000}'
# 以":"分割输出，$1表示第一个，依次递增,{print(($1 * 3600) + ($2 * 60 + $3)) * 1000}输出计算出的值
~~~
